# Quick Cookie Export Guide

## Easiest Way: Cookie-Editor Extension

1. Install: https://chrome.google.com/webstore/detail/cookie-editor/hlkenndednhfkekhgcdicdfddnkalmdm
2. Go to x.com (logged in)
3. Click extension → Export → JSON
4. Save as `twitter_cookies.json` in this folder
5. Run: `python3 scraper_with_cookies.py`

That's it! The scraper will use YOUR login session.
